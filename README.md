# 토익 단어장
flask + vue.js로 만든 토익 단어장입니다.

## 💻 소개
토익 단어 암기를 돕는 사이트 

## 🛞 개발 환경

#### Client
- Vue.js
- Bootstrap Vue

#### Server
- Flask
- Flask_restx -> RestFul 

#### DB
- MySQL

## 🐥 목차
#### 단어
- 추가한 단어를 볼 수 있습니다.
- 추가 / 수정 / 삭제 기능이 있습니다.

#### 숙어
- 추가한 숙어를 볼 수 있습니다.
- 추가 / 수정 / 삭제 기능이 있습니다.

#### 테스트
- 단어와 숙어 10개를 랜덤으로 보여줍니다
- 테스트는 단어와 뜻으로 나뉩니다.
